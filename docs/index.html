プライバシーポリシー

1.はじめに

Masaki Sugita（以下「当方」）は、まいにちわんこ　（以下「当アプリ」)がお預かりする個人情報の管理に細心の注意を払い管理いたします。

2.取り扱う個人情報について

当アプリで扱う個人情報とは、お客様のメールアドレス、パスワード、利用状況解析のためのGoogle Firebase Analyticsの情報を指します。

取得する情報、利用目的、第三者への提供等の詳細につきましては、以下のプライバシーポリシーのリンクよりご確認ください。

Firebase Analytics(Google Inc)

3.個人情報の利用について

当方は、当アプリが取得する個人情報について、以下の目的のために利用いたします。
  • 当アプリのログイン認証のため
  • 当アプリ利用者のお問い合わせ対応のため
  • 当アプリ利用者へのご案内やお知らせをお届けするため
  • デバイスやアプリバージョンの使用状況、特定の機能の使用状況などを解析し、当アプリサービスを改善するため

4.個人情報の第三者提供の開示・提供の禁止ついて

当方は、お客様よりお預かりした個人情報を適切に管理し、次のいずれかに該当する場合を除き、個人情報を第三者に開示いたしません。

  • お客さまの同意がある場合
  • 法令に基づき開示することが必要である場合

5.個人情報の管理およびセキュリティについて

当方は、個人情報を正確かつ安全に管理します。 当方は、個人情報の紛失、破壊、改ざんまたは漏えい等を防止するため、安全対策を講じ個人情報の厳重な管理を行ないます。

6. 法令、規範の遵守と改定

当方は、保有する個人情報に関して適用される日本の法令、その他規範を遵守致します。また、当ポリシーの内容を必要に応じて見直し、その改善に努めます。
